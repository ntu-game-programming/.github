## 3. Commit Message 格式

[type]: something detail \
ex. refactor: improve code readability

| type           | description                        |
|-----------------|----------------------------|
| feat  |	add feature|
| fix   |	bug fix |
| docs | 	documentation |
| style |	formatting, missing semi colons… |
| refactor | 	rewrite codes |
| chore |	maintain, .gitignore |
| env |	environment setup or change |
